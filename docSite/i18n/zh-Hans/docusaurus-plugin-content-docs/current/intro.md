# FastGpt 是什么？

FastGPT 是一个基于向量 + LLM 模型的 AI 问答系统。可以帮助你快捷、高效的将内部文档进行训练。提供了丰富的调试工具便于调整数据，并可以通过 Iframe 或 API 的形式接入到你现有的应用中。
